Project Introduction: Revision 881

Welcome to Open881. This project aims to preserve and emulate RuneScape 3 as it existed in June 2017.

Revision 881 is a critical snapshot in RS history because it captures the game immediately following the release of the Menaphos Expansion. This was a unique era where the original Java client was still fully supported alongside the newer NXT engine.
