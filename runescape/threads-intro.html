Project Introduction: Revision 881

Welcome to Open881. This project aims to preserve and emulate RuneScape 3 as it existed in June 2017.

Revision 881 is special because it captures the game immediately following the release of the Menaphos Expansion. This was a "golden age" for the NXT client, balancing the Evolution of Combat with modern graphics and mid-level content.
