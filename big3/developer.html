Current Project

SMB Assessments CRM (in development) — a focused, offline-ready client and job tracker built for restoration and assessment work.

  • Offline first: Runs entirely in the field, syncs when back online. Uses local JSON storage for data ownership.
  • Clear documentation: Tracks clients, sites, rooms, moisture logs, and photos in a single timeline.
  • Data freedom: No vendor lock-in or hidden dependencies. Export and backup in open formats.

Future updates include permissions, printable scopes, and visual analytics. No public repository yet — still refining the schema and interface.
