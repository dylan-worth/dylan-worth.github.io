Pre-Installation Preparation


Check out my other guide on how to flash an OS onto a USB drive if you need assistance. In this guide, we're focusing on dispelling the myth that Arch is an arcane, runic mystery that only the super-secret few use to escape the Lizard Overlords.

Initial Setup


To begin, we need a working network connection. When you first boot up Arch Linux, you'll see a terminal interface. Type iwctl to launch the wireless command-line tool.

Follow these steps to connect to WiFi:
1. station wlan0 scan - Search the ethereal plane for WiFi signals.
2. station wlan0 get-networks - List the mystical WiFi energies available.
3. station wlan0 connect [Your-WiFi-Network] - Provide the sacred passphrase (your WiFi password) to connect.

Installation Process


Now that you're online, let's install Arch. There's a built-in `archinstall` script, but let's use the trendier, aptly-named `archinstall`.

1. Update the package database: pacman -Syy
2. Install `archinstall`: pacman -S archinstall
3. Run the installer by typing: archinstall

Configuration Overview


The installer will guide you through several configuration steps:

1. Keyboard Layout - Adjust this if you aren't using an English layout.
2. Disk Partitioning - Divide your drives into partitions. You can opt for automatic partitioning to wipe all drives or choose manual if dual-booting (though this guide doesn't cover that). Choose a filesystem, like `ext4`, and decide whether to use a separate folder for home.
3. Bootloader - Choose `grub-install` for a reliable bootloader and consider customizing with themes like Minegrub.
4. Hostname - Set your computer's hostname (default is `archlinux`).
5. Root Password - Set a password for the root account.
6. User Account - Create a user account, set a username (like "Dylan"), a secure password, and grant superuser privileges if desired.
7. Profile - Choose a Desktop Environment (DE) and appropriate graphics drivers (e.g., AMD).
8. Audio - Pick an audio system like `pulseaudio` or `pipewire`.
9. Kernel - Leave this on the default unless you're confident about other choices.
10. Additional Packages - Install extras like `htop`, `neofetch`, and `firefox`.
11. Network - Use `NetworkManager` to manage your connection.
12. Repositories - Enable `multilib` for additional repository support.

Once everything is configured, click "Install." The installation process will take some time. After it is complete, reboot your computer and login to the account you created earlier.
