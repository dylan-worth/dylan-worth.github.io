Class AttackOpPriority

The AttackOpPriority enum defines the client-side logic for rendering the "Attack" interaction in the context menu. This directly impacts how we handle user input for PVP and PVM combat scenarios.

Technician Note: We use these constants to map the varp (player variable) settings coming from the server. When a player changes their "Attack Options" in the UI, we update the active priority instance to modify menu entry sorting.
